Nowe możliwości CSS w porównaniu z HTML

Nowe możliwości CSS przynoszą wiele korzyści w porównaniu z tradycyjnym formatowaniem HTML:

  1. Wycofywanie przestarzałych atrybutów HTML: Przestarzałe atrybuty i znaczniki w HTML, które dotyczą formatowania, stopniowo są zastępowane przez deklaracje CSS.
  2. Przejrzystość i krótsze dokumenty: Dokumenty z wykorzystaniem CSS są zazwyczaj bardziej przejrzyste i krótsze.
  3. Zarządzanie stylami: CSS pozwala na łatwe zarządzanie formatowaniem całej serii dokumentów poprzez stosowanie zewnętrznych arkuszy stylów.
  4. Stosowanie klas selektorów: Dzięki klasom selektorów możliwe jest określenie wielu atrybutów formatowania w jednym miejscu, co przyspiesza modyfikację strony.
  5. Różnorodność jednostek i sposobów definiowania kolorów: CSS oferuje możliwość stosowania różnych jednostek oraz sposobów definiowania kolorów.
  6. Różne wartości pogrubienia czcionki: CSS umożliwia wybór spośród różnych wartości pogrubienia czcionki.
  7. Dodatkowe możliwości formatowania tekstu: CSS oferuje szereg dodatkowych możliwości formatowania tekstu.
  8. Efektowne filtry graficzne: CSS pozwala na stosowanie efektownych filtrów graficznych.
  9. Dodatkowe możliwości określenia wyglądu dokumentów podczas ich drukowania: CSS umożliwia określenie wyglądu dokumentów podczas ich drukowania.

Uwaga: CSS 3 oferuje jeszcze więcej rozwiązań. Aby dowiedzieć się więcej, sprawdzaj stronę organizacji W3C oraz najnowsze specyfikacje.
