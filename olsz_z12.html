Rodzaje jednostek HTML I CSS

HTML - Najczęściej używane jednostki

Jednostka Opis
px Piksele - jednostka absolutna, niezależna od rozdzielczości ekranu.
% Procenty - jednostka relatywna do wielkości rodzica.
em Jednostka odpowiadająca aktualnej wielkości czcionki.
rem Jednostka odpowiadająca wielkości czcionki elementu korzenia (root).
vh Wysokość widoku - jednostka relatywna do wysokości okna przeglądarki.

CSS - Najczęściej używane jednostki

Jednostka Opis
px Piksele - jednostka absolutna, niezależna od rozdzielczości ekranu.
% Procenty - jednostka relatywna do wielkości rodzica.
em Jednostka odpowiadająca aktualnej wielkości czcionki.
rem Jednostka odpowiadająca wielkości czcionki elementu korzenia (root).
vw Szerokość widoku - jednostka relatywna do szerokości okna przeglądarki.
  • Selektory proste:
    • Selektory oparte na nazwie elementu HTML.
    • Przykład: p { color: red; }
    • Stylizuje wszystkie elementy p.
  • Selektory złożone:
    • Selektory oparte na relacji między elementami HTML.
    • Przykład: div p { font-size: 16px; }
    • Stylizuje wszystkie elementy p, które znajdują się wewnątrz elementów div.
  • Selektory atrybutów:
    • Selektory oparte na atrybutach elementów HTML.
    • Przykład: a[href^="https://"] { color: blue; }
    • Stylizuje wszystkie linki z atrybutem href zaczynającym się od "https://".
