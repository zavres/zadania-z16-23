Rodzaje selektorów CSS

  • Selektor potomka:
    • Określa elementy, które są potomkami innego elementu.
  • Selektor dziecka:
    • Określa elementy, które są bezpośrednimi potomkami innego elementu.
  • Selektor braci:
    • Stosuje się do elementów, które sąsiadują w strukturze z innymi elementami na tym samym poziomie hierarchii.
  • Selektor atrybutu:
    • a) Prosty selektor atrybutu: Stosuje się do elementów z określonym atrybutem. 
      a[href] {
  /* style */
}
    • b) Selektor atrybutu o określonej wartości: Stosuje się do elementów z określoną wartością atrybutu. 
      input[type="text"] {
  /* style */
}
    • c) Selektor atrybutu zawierający określony wyraz: Stosuje się do elementów z atrybutem, który zawiera określony tekst. 
      img[alt*="obraz"] {
  /* style */
}
  • Selektory specjalne:
    • Specjalne selektory, takie jak ::first-line i ::first-letter, używane w określonych kontekstach.
  • Selektor klasy:
    • Stosuje się do elementów, które posiadają określoną klasę.
  • Pseudoklasy:
    • Specjalne przypadki selektorów, które pozwalają na zastosowanie stylów w zależności od stanu elementu lub interakcji użytkownika.
  • Selektory pseudoelementów:
    • Pozwalają na stylizację określonych części elementów, takich jak pierwsza linia tekstu lub pierwsza litera.
